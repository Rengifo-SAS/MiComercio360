# Instrucciones para Imagen de Open Graph

Para completar la implementación de SEO, necesitas crear una imagen para compartir en redes sociales.

## Especificaciones de la Imagen

- **Nombre del archivo:** `og-image.jpg` (o `og-image.png`)
- **Ubicación:** `/public/og-image.jpg`
- **Dimensiones:** 1200x630 píxeles (ratio 1.91:1)
- **Formato:** JPG o PNG
- **Tamaño máximo:** 8MB (recomendado: menos de 1MB)

## Contenido Sugerido

La imagen debe incluir:
- Logo de MiComercio360
- Título: "MiComercio360"
- Subtítulo: "Sistema POS Moderno para Empresas Colombianas"
- Colores de marca consistentes
- Diseño limpio y profesional

## Herramientas Recomendadas

Puedes crear la imagen usando:
- Canva (https://www.canva.com) - Plantilla: "Open Graph Image"
- Figma
- Adobe Photoshop
- Herramientas online como OG Image Generator

## Verificación

Una vez creada la imagen, verifica que:
1. El archivo esté en `/public/og-image.jpg`
2. La URL sea accesible en: `https://pos.numercia.com/og-image.jpg`
3. Puedes probar la imagen usando:
   - Facebook Debugger: https://developers.facebook.com/tools/debug/
   - Twitter Card Validator: https://cards-dev.twitter.com/validator
   - LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
